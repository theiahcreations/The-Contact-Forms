# Changelog

All notable changes to The IAH Creations Contact Forms project will be documented in this file.

## [1.0.0] - 2024-12-19

### Added
- Initial release of Google Apps Script for contact forms
- Dynamic form creation with conditional branching
- Google Sheets integration for automatic response collection
- Comprehensive pricing structure for all services
- Client information collection system
- Timeline and content status tracking

### Features
- **Service Categories**: Website, Web App, Mobile App development
- **Pricing Tiers**: Basic, Medium, Advanced options for each service
- **Conditional Logic**: Form sections change based on service selection
- **Auto-Integration**: Responses automatically saved to Google Sheets
- **Professional UI**: Progress bar and section headers for better UX

### Technical Details
- Built with Google Apps Script API
- Uses FormApp for form creation
- SpreadsheetApp integration for data collection
- Conditional page breaks for dynamic user experience

### Pricing Structure
- Website services: ₹4,999 - ₹59,999 ($60 - $720)
- Web App services: ₹24,999 - ₹2L+ ($300 - $2,500+)
- Mobile App services: ₹29,999 - ₹2.5L+ ($360 - $3,000+)
